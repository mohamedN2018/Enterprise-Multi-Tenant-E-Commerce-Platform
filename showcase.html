التميّز الهندسي

هندسة تحت السطح تستحق الثقة

01

عزل متعدد المستأجرين

كل صف بيانات مرتبط بمتجره، ويُفرض العزل تلقائيًا عبر Middleware ومديري استعلام مُقيّدين — استحالة تسرّب بيانات بين المتاجر.

02

معمارية نظيفة

فصل واضح Models → Repositories → Services → Views. منطق العمل كله في طبقة خدمات، وواجهات رفيعة قابلة للاختبار.

03

غلاف استجابة موحّد

كل استجابات الـ API بصيغة ثابتة {success, message, data, errors} مع أكواد أخطاء واضحة.

04

عقد موحّد للنماذج

مفاتيح UUID، طوابع زمنية، حذف ناعم، وطوابع تدقيق على كل نموذج — مع قابلية رصد بمعرّف طلب فريد في السجلّات.

الأمان والجودة

مبنيّة آمنة من الأساس

JWT مصادقة بتدوير التوكِن وقائمة سوداء وخاصية "تذكّرني".
Argon2 تجزئة كلمات المرور بالمعيار الأقوى حاليًا.
Fernet تشفير المفاتيح الحساسة عند التخزين.
Throttle تحديد معدّل الطلبات لكل نقطة حساسة.
SSRF حماية عند الاتصال بالخدمات الخارجية.
RBAC أدوار وصلاحيات دقيقة لكل قسم.
+451 اختبار آلي يغطّي منطق العمل الحرج.
Devices إدارة جلسات الأجهزة وإلغاء الوصول.
OpenAPI توثيق تلقائي لكل نقاط الـ API.
